# Show

A modern presentation tool that rethinks how presentations should work.

Show provides a fresh approach to creating presentations, free from the legacy constraints of traditional presentation software. Focus on your content and message rather than wrestling with formatting.
Not a list of slides, but a timeline of content.
Avoid bulletpoints and death by PowerPoint and create engaging presentations that captivate your audience.

**Key Features:**
- Clean, distraction-free presentation creation
- Modern web-based interface
- Mobile-first design
- Show presentations on any device with a web browser
- Lightweight and efficient
- Data stored via Fleabox

Part of the Fleaoffice Suite - a modern, web-based suite of personal productivity tools.
