# Show

A modern presentation tool that rethinks how slides should work.

Show provides a fresh approach to creating presentations, free from the legacy constraints of traditional presentation software. Focus on your content and message rather than wrestling with templates and formatting.

**Key Features:**
- Clean, distraction-free presentation creation
- Modern web-based interface
- Mobile-first design
- Lightweight and efficient
- Data stored locally via Fleabox

Part of the Fleaoffice Suite - a modern, web-based suite of personal productivity tools.
