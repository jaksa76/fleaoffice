# List

A modern alternative to spreadsheets for managing structured data.

Instead of fighting with tables and formulas, List lets you create collections of items with custom fields. Perfect for managing inventories, contact lists, project resources, or any data that doesn't need the complexity of a full spreadsheet.

**Key Features:**
- Create lists of items with custom fields
- Simple, intuitive interface
- Mobile-first design
- Lightweight and fast
- Data stored locally via Fleabox

Part of the Fleaoffice Suite - a modern, web-based suite of personal productivity tools.
